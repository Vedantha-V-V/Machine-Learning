- **Supervised Learning**
  - Linear Regression
  - Logistic Regression
  - Decision Trees

  - Support Vector Machines

  - Confusion Matrix
  - ROC & AUC

- **Other Topics**
  - Ensemble Learning
  - Data Preprocessing
  - Data Preparation
